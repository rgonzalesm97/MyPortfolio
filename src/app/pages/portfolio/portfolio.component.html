WevDebWorld Blog:

TypeScript Angular Express Bootstrap MongoDB

WDW is a pre-design of a blog. It has all the functionalities such as article search, article CRUD, authentication and authorization for the frontend. And as for the backend, it is made with express with connection to a non-relational database such as Mongo DB. For authorization, bcrypt was used and JWT (Json Web Token) was used for authentication.

Linked List Visualizer:

TypeScript Angular Bootstrap Primeng Animations

This web application shows in an animated and didactic way how a linked list works. You can add or remove nodes or even create a new linked list and each function in it will be animated. To build it, angular was used together with bootstrap and primeng for the styles. For the animations, a powerful tool that angular offers us was used, which is angular animations, which is very versatile and dynamic, at first it may seem complex but once you get the hang of it you will feel the power of animations in your hands.
